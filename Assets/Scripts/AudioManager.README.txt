Audio Manager System
=====================

Этот скрипт предоставляет полнофункциональную систему управления аудио в Unity проекте.

Функциональность:
---------------

1. Воспроизведение фоновой музыки с плавными переходами
2. Воспроизведение звуковых эффектов
3. Управление громкостью музыки и звуковых эффектов
4. Поддержка 3D звука
5. Работа в качестве singleton (один экземпляр на всю игру)

Установка:
---------

1. Создайте новый GameObject в сцене
2. Добавьте к нему скрипт AudioManager.cs
3. Создайте префаб из этого объекта и поместите его в папку Prefabs/Audio
4. Добавьте префаб на сцену

Использование:
------------

// Воспроизведение музыки
AudioManager.instance.PlayMusic("название_музыки");

// Воспроизведение звукового эффекта
AudioManager.instance.PlaySFX("название_эффекта");

// Воспроизведение звукового эффекта в определенной позиции (3D звук)
AudioManager.instance.PlaySFX("название_эффекта", позиция);

// Установка громкости
AudioManager.instance.SetMusicVolume(0.5f); // 50% громкости
AudioManager.instance.SetSFXVolume(0.8f);   // 80% громкости

// Управление музыкой
AudioManager.instance.StopMusic();
AudioManager.instance.PauseMusic();
AudioManager.instance.ResumeMusic();

// Проверка состояния
bool isPlaying = AudioManager.instance.IsMusicPlaying();

Структура файлов:
---------------

Папка Resources должна содержать:
- Resources/Music/ - для музыкальных треков
- Resources/SFX/ - для звуковых эффектов

Поддерживаемые форматы:
---------------------

- MP3
- WAV
- OGG

Расширение функциональности:
--------------------------

Для добавления новых аудио файлов:
1. Поместите их в соответствующие папки в Resources
2. Они будут автоматически загружены при запуске игры

Для использования AudioMixer:
1. Создайте AudioMixer в папке Resources/Audio
2. Создайте группы для Music и SFX
3. Назначьте эти группы в инспекторе AudioManager
