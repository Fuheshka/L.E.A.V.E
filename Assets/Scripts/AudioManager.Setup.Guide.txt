Audio Manager Setup Guide
==========================

Следуйте этим шагам для настройки AudioManager в вашем Unity проекте:

1. Создание Audio Mixer (опционально, но рекомендуется):
   a. В папке Resources/Audio создайте новый Audio Mixer (Create -> Audio Mixer)
   b. Назовите его "MasterMixer"
   c. Добавьте две группы в микшере:
      - "Music" (дочерняя группа от Master)
      - "SFX" (дочерняя группа от Master)
   d. Настройте экспоненциальный (Decibel) слайдер для обеих групп

2. Создание префаба AudioManager:
   a. Создайте новый пустой GameObject в сцене
   b. Назовите его "AudioManager"
   c. Добавьте к нему скрипт AudioManager.cs
   d. В инспекторе назначьте AudioMixerGroup:
      - Перетащите группу "Music" в поле musicMixerGroup
      - Перетащите группу "SFX" в поле sfxMixerGroup
   e. Создайте префаб из этого объекта (Drag and Drop в папку Prefabs/Audio)

3. Настройка сцен:
   a. Добавьте префаб AudioManager на все сцены, где требуется аудио
   b. Убедитесь, что префаб находится в корне сцены (не вложенный в другие объекты)

4. Использование AudioManager в скриптах:
   a. Для воспроизведения музыки:
      AudioManager.instance.PlayMusic("название_файла_без_расширения");
      
   b. Для воспроизведения звуковых эффектов:
      AudioManager.instance.PlaySFX("название_файла_без_расширения");
      
   c. Для управления громкостью:
      AudioManager.instance.SetMusicVolume(0.5f); // 50% громкости
      AudioManager.instance.SetSFXVolume(0.8f);   // 80% громкости

5. Примеры использования:
   a. Воспроизведение фоновой музыки при запуске игры:
      AudioManager.instance.PlayMusic("Lite Saturation - Love's Eternal Glow (Background Romantic Piano)");
      
   b. Воспроизведение звука прыжка:
      AudioManager.instance.PlaySFX("jump");
      
   c. Воспроизведение звука респауна:
      AudioManager.instance.PlaySFX("respawn");

6. Добавление новых аудио файлов:
   a. Поместите новые файлы в соответствующие папки:
      - Resources/Music/ для музыки
      - Resources/SFX/ для звуковых эффектов
   b. Файлы будут автоматически загружены при запуске игры
   c. Используйте имя файла без расширения для воспроизведения

7. Тестирование:
   a. Запустите сцену и проверьте воспроизведение аудио
   b. Проверьте работу управления громкостью
   c. Убедитесь, что звуки воспроизводятся корректно на всех сценах

Советы по устранению неполадок:
- Если аудио не воспроизводится, проверьте консоль на наличие ошибок
- Убедитесь, что AudioManager префаб добавлен в сцену
- Проверьте, что имена аудио файлов указаны правильно (без расширения)
- Убедитесь, что файлы находятся в папке Resources
